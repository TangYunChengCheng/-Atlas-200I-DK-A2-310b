三、使用说明
1.将远控昇腾板多维度目标识别.apk文件安装到手机上
2.将无线网卡插入昇腾板子上
3.手机连接无线网卡的wifi
4.运行手机app，把录屏、读取系统文件、读取相机、悬浮窗功能打开
5.板端安装uv包管理工具，解压yolo_demo.tar.gz文件夹，进入yolo_demo文件夹下，输入
source yolo_env/bin/activate/ 激活python环境
6.输入python main_ui.py
运行板端程序，探索手机app与板端程序的交互


四、项目功能
本系统分为 “昇腾 NPU 运算核心（板端）” 和 “全场景遥控器（Android 端）”，通过局域网实现了跨设备的无缝视觉推理与硬件控制。

1. 核心 AI 与视觉引擎 (PC 端)
昇腾 NPU 光速推理：深度集成 AscendModel，充分使用NPU 算力。
全景状态面板：实时监控渲染 FPS 与 NPU 底层推理耗时 (ms)。
模型热插拔：支持随时通过 UI 或手机端“加载/卸载” .om 模型，动态释放显存。
参数实时热更：AI 识别种类、置信度 (Conf) 阈值、NMS 阈值均可无缝实时调节。

2. 八大输入源全场景覆盖 (双端联动)
PC 原生四摄：一键读取本地图片、本地视频、桌面屏幕截取、USB 摄像头。

手机跨屏四摄：
手机摄像：通过后台 MJPEG 协议将手机镜头实时推流至 NPU。
手机录屏：底层 MediaProjection 抓取手机屏幕帧，跨端投屏识别。
极速图库：直接从手机相册挑选图片/视频，通过 Socket 隧道物理直连推送至 PC 进行推理。

3. 企业级网络与通信架构
UDP 雷达动态监听：后台心跳机制，手机入网瞬间识别，断网/杀后台 5 秒内自动清理死节点，支持 IP 动态去重。

网关自动穿透 (零配置)：Android 端自动侦测所连接的 PC 热点网关，自动绑定目标 IP，实现“打开即用”的无感连接。
Socket 暴力直连协议：彻底抛弃传统 HTTP 表单上传，自建 TCP Socket 128KB 缓冲区，实现局域网极速文件互传，彻底杜绝内存溢出。
极低延迟视频流：定制 RealtimeVideoCapture 消费队列，始终只取最新一帧，解决网络波动导致的画面积压假死。

4. 极致的移动端体验 (Android 端)
远程文件管理器：手机可直接请求并浏览 PC 端的系统硬盘目录，选中视频/图片后，PC 端静默加载播放，免除电脑弹窗打扰。

智能悬浮窗保活：前台操作时自动隐藏；切回桌面或熄屏时，自动弹出“黑胶囊”悬浮护盾，利用系统 WindowManager 机制防止后台断网。
沉浸式画中画 (PiP)：全面适配高版本 Android，应用退居后台时可无缝切换至画中画模式保持推流。
极客开发者后台：底部版权信息隐藏三击彩蛋，呼出带有系统环境、连接状态、以及精确到秒的倒序滚动“终端级”运行日志。
全链路状态机：所有信号操作均采取“新浪推旧浪”的无脑强杀机制，交互极其清爽丝滑，永不卡死。
